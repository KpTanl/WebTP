# 07 高频错误与常见报错

## 24. 考试高频错误与检查清单

### 高频错误

- 表单 `name` 和后端 `@RequestParam` 不一致
- 返回模板名写错
- 模板文件不在 `templates`
- 静态文件和模板文件放错目录
- 忘记写 `@Entity`
- 忘记写 `@Id`
- Repository 没继承 `JpaRepository`
- Security 下 POST 表单忘记 CSRF
- 密码比较用了 `equals`
- `Principal` 在未登录时直接使用
- `redirect:` 和直接返回页面混了
- API key 没配
- DTO 字段名和 JSON 对不上

### 提交前检查

1. 项目能不能启动
2. 首页能不能打开
3. 表单能不能提交
4. 页面能不能显示数据库数据
5. 新增后数据库里有没有记录
6. 登录能不能成功
7. 密码是不是加密保存
8. API 能不能调通
9. DTO 能不能正确接收数据
10. 每个 POST 表单有没有处理

## 25. 常见报错

### Whitelabel Error Page

- 一般原因：路由不存在、模板不存在、Controller 报异常
- 处理：先看控制台真实异常

### 404 Not Found

- 一般原因：URL 写错、没有对应映射

### 405 Method Not Allowed

- 一般原因：表单发 `POST`，后端只写了 `@GetMapping`

### Template not found

- 一般原因：返回了模板名，但 `templates` 里没有同名文件

### Failed to configure a DataSource

- 一般原因：没写数据源配置、缺 JDBC 依赖

### Cannot determine Dialect

```properties
spring.jpa.database-platform=org.hibernate.community.dialect.SQLiteDialect
```

### No suitable driver

- 一般原因：没加 `sqlite-jdbc`

### Table not found / Column not found

- 一般原因：Entity 和表结构不一致

### `UsernameNotFoundException`

- 一般原因：登录用户查不到

### `Bad credentials`

- 一般原因：用户存在，但密码匹配失败

### `NullPointerException`

- 一般原因：取到的是 `null` 但直接调用方法

### `Optional` 取值异常

- 一般原因：`optional.get()` 时没有值

### `Could not resolve placeholder`

- 一般原因：配置文件没写对应 key

### JSON 映射失败

- 一般原因：DTO 字段名和 JSON 对不上

### API 调用失败

- 一般原因：API key 错、URL 错、网络问题

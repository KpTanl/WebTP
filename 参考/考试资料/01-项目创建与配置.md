# 01 项目创建与配置

## 1. 课程总览与考试思路

主线：

`静态页面 -> Spring Boot -> 路由/表单 -> Thymeleaf -> SQLite/JPA -> 实体关系 -> 登录/加密 -> API -> 入库 -> 页面展示`

按仓库里的 TP：

- `TP1`：HTML / CSS / 表单
- `TP2`：Spring Boot 创建、路由、参数、Session、表单处理
- `TP3`：Thymeleaf 模板
- `TP4`：SQLite、JPA、CRUD、实体关系、密码加密
- `TP6`：外部 API、Spring Security、用户数据、收藏/待看

综合题通常是：

1. 创建项目
2. 配依赖
3. 配数据库
4. 写 Entity / Repository / Service / Controller
5. 接 API
6. DTO 转 Entity 入库
7. Thymeleaf 展示
8. 加登录和密码加密

## 2. 从 VS Code 创建 Spring Boot 项目

### 创建步骤

1. `Spring Initializr: Create a Maven Project`
2. 选择 `Java`
3. 选择 Spring Boot 版本
4. 填 `Group`、`Artifact`、`Name`
5. 勾选依赖
6. 导入 VS Code

### 推荐填写

- `Group`：`fr.miage`
- `Artifact`：`examdemo`
- `Name`：`examdemo`
- `Packaging`：`Jar`
- `Java`：17 或 21

### 目录结构

```text
examdemo
├── pom.xml
├── src
│   ├── main
│   │   ├── java
│   │   └── resources
│   │       ├── application.properties
│   │       ├── static
│   │       └── templates
│   └── test
```

### `static` 和 `templates`

- `static`：静态资源，浏览器可以按路径直接访问
- `templates`：Thymeleaf 模板，通常由 Controller 返回视图名后再渲染

例子：

- `src/main/resources/static/form.html`
  对应访问路径通常是 `/form.html`
- `src/main/resources/templates/member-list.html`
  Controller 里一般写 `return "member-list";`

不要混：

- `/form.html` 是路径
- `member-list` 是视图名
- `templates/member-list.html` 才是模板文件

### 启动命令

```powershell
mvn spring-boot:run
```

或者：

```powershell
.\mvnw.cmd spring-boot:run
```

## 3. `pom.xml` 常用依赖怎么选

### Initializr 可直接勾选

- Spring Web
- Thymeleaf
- Spring Data JPA
- Spring Security
- DevTools

### 需要手动补

- SQLite JDBC
- Hibernate SQLite Dialect
- `spring-security-crypto`

### 通用依赖模板

```xml
<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-thymeleaf</artifactId>
    </dependency>

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-data-jpa</artifactId>
    </dependency>

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-security</artifactId>
    </dependency>

    <dependency>
        <groupId>org.springframework.security</groupId>
        <artifactId>spring-security-crypto</artifactId>
    </dependency>

    <dependency>
        <groupId>org.xerial</groupId>
        <artifactId>sqlite-jdbc</artifactId>
        <version>3.50.3.0</version>
    </dependency>

    <dependency>
        <groupId>org.hibernate.orm</groupId>
        <artifactId>hibernate-community-dialects</artifactId>
        <version>6.4.4.Final</version>
    </dependency>

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-devtools</artifactId>
        <scope>runtime</scope>
        <optional>true</optional>
    </dependency>

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>
```

### 依赖选择规则

- 做页面：`web` + `thymeleaf`
- 做数据库：`data-jpa` + `sqlite-jdbc` + `dialect`
- 做登录：`starter-security`
- 只做加密：至少 `spring-security-crypto`
- 调 API：`starter-web`

## 4. `application.properties` 常用配置模板

### 最小 Web 项目

```properties
spring.application.name=examdemo
server.port=8090
spring.thymeleaf.cache=false
```

### SQLite + JPA

```properties
spring.application.name=examdemo
server.port=8090

spring.datasource.url=jdbc:sqlite:db/examdemo.db
spring.datasource.driver-class-name=org.sqlite.JDBC
spring.jpa.database-platform=org.hibernate.community.dialect.SQLiteDialect
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
spring.thymeleaf.cache=false
```

### 带默认账号和 API

```properties
spring.application.name=examdemo
server.port=8090

spring.datasource.url=jdbc:sqlite:movies.db
spring.datasource.driver-class-name=org.sqlite.JDBC
spring.jpa.database-platform=org.hibernate.community.dialect.SQLiteDialect
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
spring.thymeleaf.cache=false

app.security.default-user.name=admin
app.security.default-user.password=admin123

movie.api.base-url=https://api.themoviedb.org/3
movie.api-key=YOUR_API_KEY
```
